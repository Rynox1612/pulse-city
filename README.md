# Pulse City - City-Wide Emergency Healthcare Coordination Platform

Pulse City is a comprehensive healthcare coordination platform designed to help users quickly find nearby hospitals, check real-time doctor and emergency room availability, and send emergency requests directly to hospitals.

## Features
- **User Features:** Register/login, manage profiles, find nearby hospitals, search by specialty/city, and send live emergency requests.
- **AI Symptom Assistant:** Provides first-level triage and suggests immediate actions (Note: NOT a substitute for professional medical advice).
- **Hospital Admin Features:** Manage hospital profile, track doctor availability, monitor and update hospital resources (beds, ICUs, ventilators), and respond to incoming emergency requests.

## Tech Stack
- **Frontend:** React, Vite, Tailwind CSS, React Router DOM, Axios, React-Leaflet
- **Backend:** Node.js, Express.js
- **Database:** MongoDB with Mongoose
- **Authentication:** JWT, bcrypt

## Project Structure
The repository is divided into two primary sections:
- `/client`: Vite + React frontend application.
- `/server`: Node + Express backend application.
